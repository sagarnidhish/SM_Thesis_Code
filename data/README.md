# Dataset Guide

## Source workbooks

`source/flat_plate_literature_compiled.xlsx` contains 102 rows. `source/sphere_literature_compiled.xlsx` contains 282 rows. Each workbook has one sheet named `literature_compiled_data`, frozen descriptive headers, comments defining the columns, and no derived formulas.

| Column | Meaning |
|---|---|
| `reynolds_number` | Dimensionless Reynolds number, Re. |
| `nusselt_number` | Dimensionless literature-compiled Nusselt number, Nu, before the thesis synthetic perturbation. |

The files preserve the archived numerical values and row order. They include manually digitized experimental results compiled from the literature; they are not raw instrument exports.

## Thesis-processed CSV files

`thesis_processed/flat_plate_thesis_processed.csv` and `thesis_processed/sphere_thesis_processed.csv` are generated by `python scripts/prepare_data.py`.

| Column | Meaning |
|---|---|
| `source_row` | One-based data-row identifier in the corresponding source workbook (header excluded). |
| `reynolds_number` | Source Reynolds number, unchanged. |
| `source_nusselt_number` | Source Nusselt number before synthetic scatter. |
| `perturbation_factor` | Deterministic multiplicative factor applied by the thesis plotting workflow. |
| `nusselt_number` | Final analysis value: `source_nusselt_number * perturbation_factor`. |

## Known archival quirks

- The flat-plate source contains one duplicate Reynolds-number row.
- The sphere source contains nine duplicate Reynolds-number rows and nine decreases in the original row order.
- These duplicate and out-of-order observations are preserved because the purpose of the default mode is exact thesis reproduction.
- Sorting, averaging duplicate coordinates, or selecting a single experimental series may be scientifically useful, but it changes the analysis input and should be labeled as a sensitivity analysis.

## Recommended reproduction input

Use the files under `thesis_processed/` to reproduce the archived thesis workflows. They are the unambiguous, deterministic numerical tables consumed by the analysis scripts. Use the source workbooks when auditing provenance or testing the effect of synthetic perturbation.
